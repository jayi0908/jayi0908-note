# Mac Helper

> Mac 一生推！

MacBook 以出色的性能和续航成为轻薄本的高性价比选择（特别是 mba —— MacBook Air，吃国补简直性价比拉满了）. 但是 Mac 有一些（与 Windows 相比）致命的缺陷，这使得使用 Mac 会经历一些阻碍，甚至会让人觉得不爽：

- 游戏生态非常不丰富，Intel 芯片和 M 系列芯片有一些各自的解决办法，但是总体上来说 Mac 支持的游戏还是非常少的.
- 存储限制，MacBook 的存储贵的离谱，一般入手 mba 的都会选择 256GB/512GB 的版本，特别是 256GB 的版本，存储空间非常有限.
- 软件兼容性，确实有很多软件有对应的 Mac 版本，甚至有些软件在 Mac 上运行得更好，但还是有一些常用软件获取 Mac 版本的比较麻烦，甚至有一些软件没有 Mac 版本.

下面我会结合自己的使用经验，介绍一些应对这些问题的工具和方法，当然由于使用情景因人而异，以下内容仅供参考：

- [获取 Mac 软件](./software/index.md)：介绍一些获取 Mac 软件的方法和工具.
- [节约存储空间](./storage/index.md)：通过各种方法保证 Mac 的使用体验与此同时尽可能减少存储空间的占用.
- [在 Mac 上玩游戏](./game/index.md)：介绍一些在 Mac 上玩游戏的方法和工具.
